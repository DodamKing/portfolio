DIBE: 실시간 음원 스트리밍 서비스

DIBE는 최신 음원 차트를 실시간으로 제공하고 고품질 스트리밍 서비스를 제공하는 웹 애플리케이션입니다. 음악 lovers를 위한 올인원 플랫폼을 목표로, 사용자 경험을 최우선으로 고려하여 개발되었습니다.

주요 기능

  • 실시간 음원 차트: Python과 Selenium을 이용해 Top 100 음원 차트를 주기적으로 크롤링하여 최신 트렌드를 제공합니다.
  • 웹 플레이어: 재생, 일시정지, 다음곡, 재생목록, 반복재생, 셔플 등 풀 기능 웹 플레이어를 자체 개발하여 제공합니다.
  • 개인화 기능: 사용자별 플레이리스트 생성, Local Storage를 활용한 재생 정보 저장 등 개인화된 음악 경험을 제공합니다.
  • 사용자 관리: 자체 로그인 시스템과 카카오 소셜 로그인을 지원하며, 비밀번호 찾기 등 편의 기능을 제공합니다.
  • 관리자 기능: 회원 관리, 음원 정보 관리 등 서비스 운영에 필요한 관리자 기능을 구현했습니다.
Node.js Express.js MongoDB EJS Python BeautifulSoup Selenium

기술적 도전
